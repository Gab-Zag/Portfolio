## 📋 Sobre

Portfólio pessoal de **Gabriel Alves Pereira**, desenvolvedor Full Stack com foco em **Back-end Java**. O site apresenta minha trajetória, habilidades, projetos e formas de contato — construído do zero com HTML, CSS e JavaScript puro, sem frameworks.

---

## 📁 Estrutura

```
portfolio/
├── index.html          # Página principal — todas as seções
├── style.css           # Estilos globais + responsivo
├── script.js           # Animações, scroll, formulário
├── main.jpg            # Foto de perfil
├── Currículo Gabriel.pdf  # CV para download
└── README.md           # Este arquivo
```

---

## 🗂️ Seções

| # | Seção | Descrição |
|---|-------|-----------|
| 1 | **Home** | Apresentação com animação de digitação e painel de informações |
| 2 | **Services** | Serviços oferecidos — Back-end, APIs, DevOps, etc. |
| 3 | **Skills** | Barras de progresso animadas + badges de tecnologias |
| 4 | **Education** | Timeline com formação acadêmica |
| 5 | **Experience** | Cards de experiências profissionais |
| 6 | **Contact** | Formulário funcional via EmailJS + links de contato |

---

## 🛠️ Stack do Projeto

| Tecnologia | Uso |
|------------|-----|
| HTML5 | Estrutura semântica |
| CSS3 | Estilos, animações, responsivo |
| JavaScript | Interações, scroll, formulário |
| Font Awesome 6 | Ícones |
| Google Fonts | Poppins + Space Mono |
| EmailJS | Envio de emails do formulário |
| Vercel | Hospedagem e deploy contínuo |
| Java | API para envio do email |

---

## ✨ Features

- ⚡ **Animação de digitação** com múltiplos títulos rotativos
- 🎨 **Design dark** com paleta vermelho `#b74b4b`
- 📊 **Skill bars** com animação ao entrar na viewport
- 🔄 **Anel giratório** na foto de perfil
- 📱 **Totalmente responsivo** — mobile, tablet e desktop
- 🧩 **Info bar** com status, stack e atividade recente
- 📩 **Formulário de contato** com envio real via EmailJS
- 🚀 **Deploy automático** — qualquer push na `main` atualiza o site
